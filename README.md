# goinx

Multiple domain reverse proxy with golang

## Feature

- Support static server
- Support multi domain proxy
- Support HTTPS

## Usage

**By Golang**

## Build File
```bash
go install

go build //Windows

GOOS=linux GOARCH=amd64 CGO_ENABLED=0 go build // To run in alpine

```
